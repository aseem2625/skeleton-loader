### Description

A simple-scalable approach to implement skeleton loader in React.

#### Development

```
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.


#### Production Build

```
npm run build
npx serve -s build // To serve production build
```
